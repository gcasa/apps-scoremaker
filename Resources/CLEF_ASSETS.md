# Clef Assets

- `treble_clef.png`: transparent raster rendering of the treble clef glyph.
- `bass_clef.png`: transparent raster rendering of the bass clef glyph.

The previous image files came from Wikimedia score-component assets that included staff lines. These replacement PNGs contain only the clef glyphs so they can be drawn over the app's own staff lines.
